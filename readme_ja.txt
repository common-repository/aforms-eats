== Description ==

**AForms Eats** は飲食店向けの注文フォーム作成ソフトです。圧倒的な使いやすさの注文フォームを簡単に作れます。テイクアウトやデリバリーに、あなたは食事をオンラインで売れます。メニューが少なければ、今日からでも。

AForms Eatsは使いやすさに注力しており、圧倒的な使いやすさの注文フォームをカンタンに作れます。
また、AForms Eatsは多くの飲食店オーナーから頂いたたくさんの要望を取り入れていますので、あなたのお店のフォームもきっと作れます。
高度な機能が搭載された有料版もあります。

== Feature List ==

- javascriptプログラムが50KB(gzipなし)。違和感ゼロのロード
- 100%レスポンシブ・モバイルフレンドリー
- ブラウザサポート：Chrome、Firefox、Edge、IE11、iOS Safari、Android Chrome
- おすすめ、SALEなどリボンの表示
- リアルタイムの見積表示
- インスタントバリデーション（リアルタイムバリデーション）
- オンラインフォームビルダー
- 「ラベリングロジック」で分かりやすい条件分岐
- 柔軟な税の取り扱い（内税・外税、税率、端数処理）。軽減税率にも対応
- 確認画面に対応
- サンキューメール、通知メール
- 文言・通貨のカスタマイズ
- きめ細やかな権限設定による多店舗・多役職のサポート
